# Midas Core — Financial Transaction Processing System

A high-volume financial transaction processing backend built with Java and Spring Boot for JPMorgan Chase & Co.

**Project Type:** Advanced Software Engineering Simulation (FORAGE)  
**Status:** ✅ Completed  
**Date:** May 2026

---

## 📋 Overview

Midas Core is a robust backend system that processes high-volume financial transactions with real-time message ingestion, persistent data storage, and external service integration. The system demonstrates enterprise-grade architecture patterns and best practices used in production financial systems.

---

## 🎯 Key Features

- **High-Volume Transaction Processing** - Handles large-scale transaction flows
- **Real-Time Message Ingestion** - Apache Kafka integration for streaming data
- **Transaction Validation** - Comprehensive validation logic for transaction integrity
- **Balance Management** - Account balance tracking and updates
- **Incentive System** - External API integration for incentive calculations
- **Database Persistence** - H2/JPA for reliable data storage
- **REST API Integration** - External service communication
- **Unit Testing** - Comprehensive JUnit test suite

## 🏗️ Architecture

The system is built on a microservices-ready architecture with clear separation of concerns:

- **Controller Layer** - REST API endpoints
- **Service Layer** - Business logic implementation
- **Repository Layer** - Data access and persistence
- **Entity Models** - Domain objects
- **Kafka Integration** - Event-driven message processing
